Keyboard Shortcuts.


The following keys and combinations can be used as alternatives to selecting items from the menus regardless of which sub-window is active...

CTRL + N           New Project.
CTRL + O           Open Project (CTM file).
CTRL + S           Save Project As (CTM file).
ALT + O            Import Map Image (BMP, PNG files).
ALT + S            Export Map Image (BMP, PNG files).
ALT + E            Import VICE x64 Emulator Snapshot (VSF file).
CTRL + Z           Undo.
CTRL + ALT + Z     Redo.
F12                Reset window layout.

The following keys and combinations can be used as alternatives to selecting items from the menus depending on which sub-window is active...

CTRL + A            Select all.
INS                 Insert blank item into set.
DEL                 Delete selected item(s) from set.
CTRL + DEL          Clear selected item(s).
CTRL + X            Cut Selected item(s).
CTRL + C            Copy selected item(s).
CTRL + V            Paste previously cut/copied item(s).
ALT + V             Paste (Overwrite) previously cut/copied item(s).
CTRL + ALT + V      Paste (Append) previously cut/copied item(s).
CTRL + U            Scroll selected item(s) up.
CTRL + D            Scroll selected item(s) down.
CTRL + L            Scroll selected item(s) left.
CTRL + R            Scroll selected item(s) right.
ALT + L             Flip selected item(s) left-to-right.
ALT + T             Flip selected item(s) top-to-bottom.
CTRL + SHIFT + L    Reflect selected item(s) left-to-right.
CTRL + SHIFT + T    Reflect selected item(s) top-to-bottom.
CTRL + I            Negative (colour index invert) selected item(s).
ALT + B             Cycle selected items backwards.
ALT + F             Cycle selected items forwards.
F1                  Previous Char/Tile/Brush.
F2                  Next Char/Tile/Brush.
ALT + R             Reset tile name and tag.

CTRL + Mouse-Wheel  Zoom Window contents In/Out.
Mouse-Wheel         Scroll Window contents Up/Down.
